## Executive Summary

Vespr is the universal autonomous operations orchestration platform. It works for any team that has daily priorities, workflows, and deliverables: engineering teams, finance departments, HR teams, operations teams, and marketing teams.

The core loop is identical:
1. Read daily context (issues, tasks, priorities)
2. Intelligently triage (what matters most)
3. Orchestrate work (break down, assign, schedule)
4. Execute autonomously (AI handles routine work)
5. Surface for review (human approval)

**You're no longer the bottleneck.**

### Why This Matters

Operational leaders across all functions waste 30-50% of their time on:
- **Triaging** — What's most important?
- **Planning** — How do we break it down?
- **Assigning** — Who should do this?
- **Tracking** — Where are we?
- **Chasing** — Why isn't this done?

**Vespr automates all of this.** Engineering leaders save 15 hours/week. Finance teams save 12 hours/week. HR teams save 10 hours/week.

**Result:** Teams ship/complete 1.5-2x more work in same time. Leaders get their lives back.

## The Universal Loop (How It Works for Any Team)

### Engineering Example

```
Input: Daily bugs, issues, deliverables
├─ GitHub issues (47 open)
├─ Sentry errors (23 instances, frequency trending up)
├─ Deliverables due (dashboard Friday, mobile next week)
└─ Team capacity (5 engineers, 2 on-call)

Triage: AI scores all 47 issues
├─ API timeout (80 points) → rank 1
├─ Payment error (75 points) → rank 2
├─ Dashboard feature (60 points) → rank 3
└─ Tech debt (40 points) → rank 10

Decision: Leader approves top 3 priorities
Execute: AI assigns tasks, executes bugs autonomously
Review: Engineer reviews PR (5 min)
Ship: Merge → Deploy

Result: 2 bugs fixed today, dashboard on track
```

### Finance Example (Identical Loop)

```
Input: Daily invoices, reconciliations, reports
├─ QuickBooks: 47 invoices pending approval
├─ Stripe errors: 12 failed payments
├─ Expensify: 23 expense reports waiting
└─ Team capacity: 2 finance people, year-end close coming

Triage: AI scores all 47 items
├─ Large vendor invoices (80 points) → rank 1
├─ Failed payments (75 points) → rank 2
├─ Expense reports (60 points) → rank 3
└─ Reconciliation (40 points) → rank 10

Decision: Finance manager approves top 3 priorities
Execute: AI processes invoices, reconciles, generates reports
Review: Manager verifies output (10 min)
Ship: Batch payments → Close period

Result: 2 days of AP work done in hours
```

### HR/Recruitment Example (Same Loop)

```
Input: Daily recruitment, onboarding, compliance
├─ Greenhouse: 12 open reqs, 47 candidates in pipeline
├─ Slack: 3 urgent hiring requests from managers
├─ Workday: 15 onboarding tasks pending
└─ Team capacity: 1 HR person + 2 managers hiring

Triage: AI scores all items
├─ Critical open reqs (80 points) → rank 1
├─ Hot candidates (75 points) → rank 2
├─ Onboarding backlog (60 points) → rank 3
└─ Compliance training (40 points) → rank 10

Decision: HR leader approves top 3 priorities
Execute: AI sends interview requests, creates offers, schedules onboarding
Review: HR reviews offers (10 min), manager approves (5 min)
Ship: Send offers → Start onboarding

Result: 3 candidates moving forward, 5 started onboarding
```

- Engineering is hardest vertical (good test bed)
- Fastest feedback (shipping is measurable)
- Proves loop works before expanding

- Not trying to be everything (we focus on orchestration)
- Deep integrations per vertical (not shallow Zapier-like connections)
