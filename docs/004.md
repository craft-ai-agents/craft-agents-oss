 已经把“Everything Claude Code”那套框架落地到 Craft Agents 项目里，并让新建 workspace 自动带上这套结构。这样你后续开发可以直接用 /plan、/tdd、/code-review 这些流程，减少反复搭建的成本。

  改动说明

  - 新增默认模板：apps/electron/resources/workspace-template/，包含 agents/、commands/、contexts/、hooks/、rules/、skills/、mcp-configs/、plugins/ 的基础框架与示例内容。
  - 自动注入模板：packages/shared/src/workspaces/storage.ts 在创建 workspace 时，如果目录为空且设置了模板路径，就复制模板内容。
  - 设置模板路径：apps/electron/src/main/index.ts 启动时写入 CRAFT_WORKSPACE_TEMPLATE_DIR，让新 workspace 自动套用模板。

  说明

  - 只有“空目录”的新 workspace 会自动套模板，避免污染已有项目目录。
  - 已有 workspace 如果也要套用，我可以加一个“手动安装模板”的入口，或者你直接把模板目录内容复制进去也行。
